## E-Commerce Website Using MERN STACK. 

#### The main objective behind this project was to build an e-commerce website that handles both sides client and admin using the following technologies :  

 1. Mongo Db  
 2. Expressjs  
 3. Reactjs  
 4. Nodejs  
 5. Redux  

If you are logged in as an admin a button in the navbar will show up which will give you control over your products, users, and orders.
